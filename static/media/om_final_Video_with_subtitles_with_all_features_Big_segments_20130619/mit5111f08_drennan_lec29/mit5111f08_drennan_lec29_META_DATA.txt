Description
This course provides an introduction to the chemistry of biological, inorganic, 
and organic molecules. The emphasis is on basic principles of atomic and molecular 
electronic structure, thermodynamics, acid-base and redox equilibria, chemical kinetics, 
and catalysis.

This course features video lectures in the lecture notes section and a full description of 
class assignments in the assignments section.
